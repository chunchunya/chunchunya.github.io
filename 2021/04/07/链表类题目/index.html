19. 删除链表的倒数第 N 个结点

给你一个链表，删除链表的倒数第 n 个结点，并且返回链表的头结点。

进阶：你能尝试使用一趟扫描实现吗？

示例 1：

输入：head = [1,2,3,4,5], n = 2
输出：[1,2,3,5]

示例 2：

输入：head = [1], n = 1
输出：[]

示例 3：

输入：head = [1,2], n = 1
输出：[1]

来源：力扣（LeetCode）

题解

双指针

  1. dummyHead指向头指针head，将p，q指针都指向dummyHead
  2. 移动快指针q，直至p,q之间相隔的节点个数为n
  3. 同时移动p，q，直到q的指针指向null，这个时候p指针的下一个节点即为所删除的节点
  4. 将p的next指向下下个节点
